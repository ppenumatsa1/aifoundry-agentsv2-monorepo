# Tech Stack

- Python 3.11
- FastAPI + Uvicorn
- Microsoft Agents SDK (hosting/auth components)
- Azure AI Projects SDK (`AIProjectClient`)
- Azure OpenAI Responses API via Foundry project client
- Azure Identity (`DefaultAzureCredential`)
- Azure Container Apps (runtime hosting)
- Azure Bot Service + Teams channel
- Azure Monitor / Application Insights (OpenTelemetry-based telemetry)
- Bicep + azd (infra provisioning and deployment)
- pytest (unit/integration tests)
